Description

OBS: The API has had a small change which broke the search feature, the ingredient tracker and the quiz still works though.

This group project is a React-based cocktail website using the CocktailDB API. It features a front page where users can search cocktails and filter results based on ingredients they currently have. Users can keep track of their ingredients with a “My Ingredients” search and tracker. There is a cocktail details page showing ingredients and other information, and a favorites list to save cocktails. Additionally, there is a guessing game where a random cocktail is shown and the user must guess its ingredients before the timer runs out. The timer is implemented using a React hook. The website also uses Google sign-in authentication.
