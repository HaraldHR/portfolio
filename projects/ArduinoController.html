Tech Stack

Arduino C/C++, Processing, Serial Communication

Description

This group project combines Arduino microcontrollers and a game developed in Processing to create a two-player local multiplayer experience. Each player uses a custom-built Arduino game controller made from physical sensors and actuators.

The game, inspired by agar.io, was implemented in Processing and includes a basic client-server architecture. One computer runs the server and a player client, while another computer runs only a player client. The server manages game state and synchronizes gameplay over serial and network communication.

Each Arduino communicates with its respective client using serial USB, translating hardware inputs (like button presses and joystick movement) into game commands (movement).

The game also includes a feature where, by picking up a power-up, you may change what sensor your opponent has to use to play the game.
