Project: Convolutional Neural Network from Scratch

GitHub Repository

https://github.com/HaraldHR/DL_Assignments/tree/assignment3

Tech Stack

Python, NumPy, Matplotlib

Description

This project implements a Convolutional Neural Network (CNN) entirely from scratch using only NumPy. It includes the full forward and backward pass logic for convolutional layers, ReLU activations, and fully connected layers.

The network was trained on a subset of the CIFAR-10 dataset. The training uses mini-batch stochastic gradient descent, learning rate scheduling, and loss tracking. Visualizations of filters, feature maps, and training loss were generated using Matplotlib. The finished network can be found in the file Assignment3.py in the assignment3 branch.
