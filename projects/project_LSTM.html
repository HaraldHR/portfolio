Project: LSTM Neural Network

GitHub Repository

https://github.com/HaraldHR/NLP_Project

Description

This project implements a Long Short-Term Memory (LSTM) neural network using PyTorch. It is designed to generate Shakespeare plays on a character level. The LSTM:s performance was improved by implementing BPE and Word2vec. The learning process used the Adam optimizer. The performance metrics used were loss as well as the fraction of correct n-grams and correctly spelt words.

Contributions

  • Implemented base LSTM with forward and backward propagation
  • Implemented training method using Adam optimizer.
  • Wrote a grid-search function for hyperparameter tuning.
  • Developed code for performance metrics.
  • Ran tests with different hyperparameters and evaluated results.
  • Visualized data using Matplotlib.
  • Debugged issues with BPE and Word2vec implementation.
