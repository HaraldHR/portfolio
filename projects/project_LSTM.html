Project: LSTM Neural Network

GitHub Repository

https://github.com/HaraldHR/NLP_Project

Tech Stack

Python, PyTorch, NumPy, Pickle, Matplotlib

Description

This group project implements a Long Short-Term Memory (LSTM) neural network using PyTorch. It is designed to generate Shakespeare plays on a character level. The LSTM:s performance was improved by implementing BPE and Word2vec. The learning process used the Adam optimizer. The performance metrics used were loss as well as the fraction of correct n-grams and correctly spelt words. Various sampling methods were also explored.

OBS: The repository is very messy, the vanilla LSTM implementation can be found in the file model_lstm_torch.py and the version implementing BPE and Word2vec can be found in the same file but on the Tokenization branch
