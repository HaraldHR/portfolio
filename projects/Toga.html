Description

This open source contribution was made to the BeeWare Toga repository. The update enhances the `Canvas.write_text` and `Canvas.measure_text` methods by adding support for configurable line spacing.

Previously, while the methods accepted multiple lines, the spacing between lines could not be adjusted. With the new `line_height_factor` parameter, developers can now control line spacing programmatically â€” for example, by linking it to a slider in the UI, which is what we did for debugging purposes. This feature allows for improved text layout flexibility in canvas-based apps.

The change was implemented across all backends that implement a canvas:

  • Android
  • Cocoa (macOS)
  • GTK (Linux)
  • iOS
  • WinForms (Windows)
