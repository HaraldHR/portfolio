Project: 3D Voxel RPG (Unity)

GitHub Repository

https://github.com/HaraldHR/Voxel-RPG

Tech Stack

Unity, C#, MagicaVoxel

Description

This is an in-progress 3D voxel-style RPG developed using Unity and C#. The game features a fully functional player movement system (with support for jumping, walking, and rotating), as well as an inventory system with UI for picking up, storing, and using items.

Players can also interact with objects in the world using a custom-built interaction system that allows for triggering behaviors like picking up items or opening containers.

All graphics and models were created by me in MagicaVoxel to match the stylized voxel aesthetic. While development is ongoing, the core systems are in place and ready to be expanded with gameplay features such as combat, quests, and dialogue.
